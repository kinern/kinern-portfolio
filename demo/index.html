JTrack

Compact exercise tracking app using React Native, SQLite and Expo

JTrack is a Don't Break The Chain style exercise tracking application. It was originally was created as a simple daily tracker to record daily Jump Rope exercises, but grew into a versatile application.

Screen Overview

Monthly Calendar

Build exercise routines using the Don't Break The Chain method

The main part of the exercise app is the calendar. You can tap on a date to enter the number of minutes exercised for that day. After saving your exercise time, a star will show behind that day. The color of the star corresponds to the amount of time entered. This calendar is a Don't Break The Chain style tool to help you build a routine.

1-6 Month Progression Graphs

Get an alternative view of your progress using the Graph Screen

The graph page shows a monthly view of your progress. It shows a line graph with markings for each day. The arrows at the top of the screen are used to change the month viewed.
